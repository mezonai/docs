# Chủ đề
Mezon giúp bạn tránh “loạn” thông tin bằng hệ thống Chủ đề - tách cuộc trò chuyện theo chủ đề, và Tin nhắn trực tiếp - hỗ trợ nhắn tin riêng cho từng cá nhân.
### **Cách dùng Chuỗi thảo luận**

</Step>
