# Text Channel

<Image style={{ margin: '1rem 2rem' }} src={textChannelImage001} alt="Text Channel 001" />

In a Text Channel, you can explore the following key features:

* [**Canvas**](./text-channel/canvas) – Create and manage documents or notes within each channel.
* [**Threads**](./text-channel/threads) – Separate topics for clearer discussions.
* [**Members**](./text-channel/members) – View member lists and their activity status.
* [**Pinned Messages**](./text-channel/pinned-messages) – Pin important messages for easy access.
* [**Notification Settings**](./text-channel/notification-settings) – Customize how you receive notifications for each channel.
* [**Gallery**](./text-channel/gallery) – View all images shared in the conversation.
* [**Files**](./text-channel/files) – Manage all files sent in the conversation.
* [**Search**](./text-channel/search) – Quickly find messages, files, or senders.
* [**Inbox**](./text-channel/inbox) – Centralized notifications for all activities related to you.
* [**Topic**](./text-channel/topic) – Start a private discussion from a message to keep the main conversation organized and focused.
