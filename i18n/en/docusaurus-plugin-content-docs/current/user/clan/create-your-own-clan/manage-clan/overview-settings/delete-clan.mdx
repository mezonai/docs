# Delete Clan

If your Clan is no longer in use or you want to discontinue its community, you can permanently delete the Clan.

:::danger Note
This action **cannot be undone**. All data, channels, and content within the Clan will be permanently removed.
:::

### How to Delete a Clan

<Steps>
  <Step title="In the left menu, select **Delete Clan**.">
    <Image
      style={{ margin: "1rem 2rem" }}
      src={deleteClan001}
      alt="Delete Clan 001"
    />
  </Step>
  <Step title="Re-enter the Clan name to confirm.">
    <Image
      style={{ margin: "1rem 2rem" }}
      src={deleteClan002}
      alt="Delete Clan 002"
    />
  </Step>
  <Step title="Click **Delete Clan**."></Step>
</Steps>
:::danger Note Once deleted, the Clan and all associated data will be permanently
lost and **cannot be recovered**. :::
