# Files
Sharing documents is just as important as sharing images. Files allow you to send and quickly access all documents shared in a Text Channel or Thread.

### **How to Send a File:**
<Tabs>
<TabItem value="PC" label="PC">
1. Open the **Text Channel** or **Thread** where you want to share.
2. Click the **plus (+) icon** to the left of the message input box.

<Image style={{ margin: '1rem 2rem' }}  src={files001} alt="Files 001" />

3. Select the file and click **Open**.

<Image style={{ margin: '1rem 2rem' }}  src={files002} alt="Files 002" />

4. Add a description if needed, or press **Enter** to send immediately.

<Image style={{ margin: '1rem 2rem' }}  src={files003} alt="Files 003" />
</TabItem>
<TabItem value="mobile" label="Mobile">
1. Open the Text Channel or Thread you want to share in.
2. Tap the plus (+) icon on the left of the message input box.

<div align="left"><img style={{ width: '30%' }} src={filesMobile001} alt="files-mobile-001" /></div>

3. Select **File** ![files-mobile-002](./images/files-mobile-002.png) from the menu.

<div align="left"><img style={{ width: '30%' }} src={filesMobile003} alt="files-mobile-003" /></div>

4. Choose the file you want to send. Add a description if needed and tap **Send**.

<div align="left"><img style={{ width: '30%' }} src={filesMobile004} alt="files-mobile-004" /></div>
</TabItem>
</Tabs>


### **How to View Sent Files:**
<Tabs>
<TabItem value="PC" label="PC">
1. In the **Text Channel** or **Thread**, click the **paperclip icon** ![files-004](./images/files-004.png) at the top of the channel.
2. The file list will display the **sender** and **time sent**.

<Image style={{ margin: '1rem 2rem' }}  src={files005} alt="Files 005" />

3. Click **Download** to save the file to your computer.

<Image style={{ margin: '1rem 2rem' }}  src={files006} alt="Files 006" />
</TabItem>
<TabItem value="mobile" label="Mobile">
1. In the Text Channel or Thread, select **Media** from the menu.

<div align="left"><img style={{ width: '30%' }} src={filesMobile005} alt="files-mobile-005" /></div>

2. The file list will show the sender and the time sent.

<div align="left"><img style={{ width: '30%' }} src={filesMobile006} alt="files-mobile-006" /></div>

3. Tap a file to **Download** and view it on your device.
</TabItem>
</Tabs>
