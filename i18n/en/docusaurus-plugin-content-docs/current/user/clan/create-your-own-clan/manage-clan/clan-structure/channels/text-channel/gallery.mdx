# Gallery
Gallery lets you view all images shared in a Text Channel or Thread and easily filter them by date to find the content you need.

### **How to Send an Image:**
<Tabs>
<TabItem value="PC" label="PC">
1. Open the **Text Channel** or **Thread** where you want to share.
2. Click the **plus (+) icon** to the left of the message input box.

![gallery-001](./images/gallery-001.png)

3. Select the image file and click **Open**.

![gallery-002](./images/gallery-002.png)

4. Add a description if needed, or press **Enter** to send immediately.

![gallery-003](./images/gallery-003.png)
</TabItem>
<TabItem value="mobile" label="Mobile">
1. Open the **Text Channel** or **Thread** you want to share in.
2. Tap the plus (+) icon on the left of the message input box.

<div align="left"><img style={{ width: '30%' }} src={galleryMobile001} alt="gallery-mobile-001" /></div>

3. Select the image you want to send.

<div align="left"><img style={{ width: '30%' }} src={galleryMobile002} alt="gallery-mobile-002" /></div>

4.  Add a description if needed, or tap the arrow to send immediately.

<div align="left"><img style={{ width: '30%' }} src={galleryMobile003} alt="gallery-mobile-003" /></div>
</TabItem>
</Tabs>


### **How to View Images:**
<Tabs>
<TabItem value="PC" label="PC">
1. In the **Text Channel** or **Thread**, click the **photo icon** ![gallery-004](./images/gallery-004.png) at the top of the channel, then select the image you want to view.

![gallery-005](./images/gallery-005.png)

2. You can **download**, **rotate**, and **zoom in/out** while viewing an image in detail.

![gallery-006](./images/gallery-006.png)

3. Select **Date Sent** to filter images by the sending date.

![gallery-007](./images/gallery-007.png)

4. Click **Apply**, and images from the selected time period will be displayed.

![gallery-008](./images/gallery-008.png)
</TabItem>
<TabItem value="mobile" label="Mobile">
1. In the Text Channel or Thread, select **Media** from the menu.

<div align="left"><img style={{ width: '30%' }} src={galleryMobile004} alt="gallery-mobile-004" /></div>

4. All images from the channel or thread are organized by date, making it easy to review and find the content you want.

<div align="left"><img style={{ width: '30%' }} src={galleryMobile005} alt="gallery-mobile-005" /></div>
</TabItem>
</Tabs>
