# Gallery
Gallery lets you view all images shared in a Text Channel or Thread and easily filter them by date to find the content you need.

### How to Send an Image
<Tabs>
<TabItem value="PC" label="PC">
<Steps>
<Step title="Open the **Text Channel** or **Thread** where you want to share."></Step>
<Step title="Click the **plus (+) icon** to the left of the message input box.">
<Image style={{ margin: '1rem 2rem' }}  src={gallery001} alt="Gallery 001" />
</Step>
<Step title="Select the image file and click **Open**.">

<Image style={{ margin: '1rem 2rem' }}  src={gallery002} alt="Gallery 002" />
</Step>
<Step title="Add a description if needed, or press **Enter** to send immediately.">

<Image style={{ margin: '1rem 2rem' }}  src={gallery003} alt="Gallery 003" />
</Step>
</Steps>
</TabItem>
<TabItem value="mobile" label="Mobile">
1. Open the **Text Channel** or **Thread** you want to share in.
2. Tap the plus (+) icon on the left of the message input box.

<div align="left"><img style={{ width: '30%' }} src={galleryMobile001} alt="gallery-mobile-001" /></div>

3. Select the image you want to send.

<div align="left"><img style={{ width: '30%' }} src={galleryMobile002} alt="gallery-mobile-002" /></div>

4.  Add a description if needed, or tap the arrow to send immediately.

<div align="left"><img style={{ width: '30%' }} src={galleryMobile003} alt="gallery-mobile-003" /></div>
</TabItem>
</Tabs>

<Image style={{ margin: '1rem 2rem' }}  src={gallery005} alt="Gallery 005" />
