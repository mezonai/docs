# Topic
Topic in chat allow you to create a separate discussion directly from a message, keeping the main conversation thread clear and organized.

### **How to Add a Topic in Chat:**
<Tabs>
<TabItem value="PC" label="PC">
1. In a **Text Channel** or **Thread**, right-click the message you want to discuss.
2. Select **Topic Discussion**

<Image style={{ margin: '1rem 2rem' }}  src={topic001} alt="Topic 001" />

3. Enter your message in the **Topic window** on the right to start the discussion.

<Image style={{ margin: '1rem 2rem' }}  src={topic002} alt="Topic 002" />

With **Topic**, you can maintain a focused main conversation while allowing detailed discussions without confusion.
</TabItem>
<TabItem value="mobile" label="Mobile">
1. In a Text Channel or Thread, press and hold the message you want to discuss further.
2. Select **Topic Discussion**.

<div align="left"><img style={{ width: '30%' }} src={topicMobile001} alt="topic-mobile-001" /></div>

3. Enter your message in the Topic window to start the exchange.&#x20;

<div align="left"><img style={{ width: '30%' }} src={topicMobile002} alt="topic-mobile-002" /></div>

### **How to review Topics you've participated in:**

* In the chat window, click **View Topic** to open the content and see the entire related discussion.

<div align="left"><img style={{ width: '30%' }} src={topicMobile003} alt="topic-mobile-003" /></div>

* Or go to **Notifications → Topics**, where all the Topics you’ve joined or followed in the Clan are displayed.

<div align="left"><img style={{ width: '30%' }} src={topicMobile004} alt="topic-mobile-004" /></div>

:::tip
With **Topic**, you can keep the main conversation focused while branching out detailed discussions without confusion.
:::
</TabItem>
