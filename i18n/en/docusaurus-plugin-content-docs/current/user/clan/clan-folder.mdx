# Clan Folder
When you join multiple Clans, the left-hand list can become long and hard to manage. Clan Folder helps you group related Clans together, keeping your interface tidy and easy to navigate.

<Tabs>
<TabItem value="PC" label="PC">
### Creating a Folder

<Steps>
<Step title="On the main Mezon interface, select the Clan you want to put into a folder.">

<Image style={{ margin: '1rem 2rem' }}  src={clanFolder001} alt="Clan Folder 001" />
</Step>
<Step title="Drag that Clan onto another Clan. When you see a green dashed rectangle, release the mouse.">
<Image style={{ margin: '1rem 2rem' }}  src={clanFolder002} alt="Clan Folder 002" />
</Step>
<Step title="The folder containing your Clans will appear.">
<Image style={{ margin: '1rem 2rem' }}  src={clanFolder003} alt="Clan Folder 003" />
</Step>
</Steps>
### Removing a Clan from a Folder

<Steps>
<Step title="Open the folder containing your Clans.">

<Image style={{ margin: '1rem 2rem' }}  src={clanFolder004} alt="Clan Folder 004" />
</Step>
<Step title="Hover over the Clan you want to remove and click the **X** icon.">

<TabItem value="mobile" label="Mobile">
<Heading as="h3">How to Create a Clan Folder</Heading>

1. On the main Mezon interface, **click and hold** the Clan you want to add to a folder.
2. **Drag the Clan onto another Clan**. When you see a square frame showing the two Clans selected, release to group them.

<div align="left"><img style={{ width: '10%' }} src={clanFolderMobile001} alt="clan-folder-mobile-001" /></div>

4. A **new Clan folder** will appear in the left-hand menu along with your other Clans.

<div align="left"><img style={{ width: '10%' }} src={clanFolderMobile002} alt="clan-folder-mobile-002" /></div>

<Heading as="h3">How to Remove a Clan from a Folder</Heading>

1. Open the folder containing your Clans.
2. Click the **X icon** on the Clan you want to remove.

<div align="left"><img style={{ width: '10%' }} src={clanFolderMobile003} alt="clan-folder-mobile-003" /></div>

3. After removal, the Clan will appear again outside the folder in your Clan list.

<div align="left"><img style={{ width: '10%' }} src={clanFolderMobile004} alt="clan-folder-mobile-004" /></div>
</TabItem>
